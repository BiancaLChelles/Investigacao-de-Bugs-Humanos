# Manual de Lógica Assistiva

> **"Usarei minhas sombras para iluminar o código e minhas falhas para construir sistemas."**

Este repositório é um sistema de **Lógica Assistiva** desenvolvido para atuar onde os sensores naturais do sistema operacional humano apresentam falhas de interocepção ou processamento. Como alguém com **perfil neurodivergente (TEA, SD/HA e TAG)**, utilizo a programação como uma ferramenta de suporte para traduzir estados internos em instruções acionáveis.


##  Contexto 
Pessoas com o perfil neurodivergente frequentemente apresentam dificuldades em notar sinais básicos do hardware (corpo), como fome ou cansaço, ou lidam com processos de software (mente) que entram em loops infinitos. Este projeto visa documentar e automatizar a detecção desses "bugs" para garantir a manutenção da performance e do bem-estar, auxiliando quem, como eu, precisa de uma "mãozinha" para funcionar.
